## Core Features

- **Music Streaming** - Discover and stream music
- **Social Network** - Follow artists, share tracks
- **NFT Marketplace** - Buy/sell music NFTs with 32 tokens
- **WIN-WIN Rewards** - Earn OGUN from streams
- **Multi-Wallet** - Connect MetaMask, WalletConnect, Coinbase

## Networks

- **Primary:** Polygon Mainnet
- **Cross-Chain:** ZetaChain integration
